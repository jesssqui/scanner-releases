# Scanner — Releases

Public download + auto-update feed for **Scanner** (a macOS bulk trading-card scanner).
Application source is maintained privately; this repo hosts only release artifacts.

- **Download:** see [Releases](../../releases/latest) → `Scanner-<version>.dmg`
- **Auto-update feed:** [`appcast.xml`](appcast.xml) (consumed by the app via Sparkle)

First install is a one-time right-click → Open (the app is self-signed); updates after that are automatic.
